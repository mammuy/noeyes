# Twinkle Tray Private

A minimalist, privacy-focused utility to manage external and internal monitor brightness via DDC/CI and WMI on Windows. 

All analytics, telemetry, troubleshooting bridges, auto-updaters, network listeners, unused languages, complex hotkeys, UI bloat, and external network requests have been removed for a clean, private, functional core.

## Building

Uses GitHub Actions to automatically generate a standalone executable.
